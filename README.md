Suivi des taches
====================
##Principales fonctionnalités
### Onglet "Liste"
* Voir la liste des taches
* Ajouter une taches
### Onglet "Travail"
* Voir la tache n°1
* Lancer le travail
* Un timer de 20 min apparait
* Possibilité de terminer la tache
* A la fin des 20 min, si la tache n'est pas finie, on affiche la raison
